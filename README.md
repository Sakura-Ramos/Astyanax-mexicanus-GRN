# Homology based reconstruction of gene regulatory networks on Astyanax mexicanus 

En este trabajo se reconstruyeron las redes de regulaci√≥n, asociadas con genes relacionados con el desarrollo del ojo, en los ecotipos de superficie y de cueva de A. mexicanus, tomando como referencia 3 organismos con interacciones reguladoras bien conocidas. 

### Prerequisites

Este trabajo se construyo utilizando: 
- [Jupyter notebook](https://jupyter.org/)
- [Python](https://www.python.org/) - Programming language (Additionally, various libraries were used such as pandas or networkx)
- [Gephi](https://gephi.org/) - Visualizer and network analysis
- [Proteinortho](https://gitlab.com/paulklemm_PHD/proteinortho) - Tool to detect orthologous genes

### Folder description

Different folders were created to organize this work. Below is a description of the contents: 
- NCBI: proteomes used in this work in fasta format for models and Astyanax.
- Proteinortho: Proteinortho outputs for model and astyanax organism.
- scripts: notebooks to recreate this project. 
- analysis: analysis for the models (hubs).
